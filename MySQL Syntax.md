### CRUD
데이터 베이스가 가지고 있는 4가지 주요 작업

1. Create
2. Read
3. Update
4. Delete

------

### db

```
- test db 생성 : CREATE DATABASE test;
- db 보기 : SHOW DATABASES; 
- test db 선택 : USE test;
- test db 삭제 : DROP DATABASE users;
```

### table
```
- table 생성 : CREATE TABLE info (id INT(11) NOT NULL)
- table 보기 : SHOW TABLES;
- info table 정보 보기 : DESK info;
```
